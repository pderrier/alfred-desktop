## Privacy

Alfred Desktop sends **no personal data** to the Alfred enrichment API — only public market identifiers (ticker symbols, ISIN codes, company names). See [README — Privacy section](README.md#privacy--what-is-sent-to-the-api) for details.

Portfolio data (positions, quantities, account names) is sent to **OpenAI** as part of LLM analysis prompts. This is inherent to the AI-powered analysis feature and disclosed in the README.

No telemetry, analytics, or tracking is included in the application.

## Reporting security issues

If you discover a security vulnerability, please report it privately via [GitHub Security Advisories](https://github.com/pderrier/alfred-desktop/security/advisories/new) or email pierre.derrier@gmail.com.
